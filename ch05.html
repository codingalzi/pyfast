문자열 '' 와 문자의 ' ' 차이점

빈 문자열empty string '' 는 아무런 문자를 포함하지 않으며 따라서 문자열의 길이, 즉 문자열에 포함된 문자의 개수가 0이다. 반면에 ' ' 는 눈에 보이지는 않지만 공백 문자 하나를 포함하는 문자열이며 길이가 1이다.
